# Adobe認定制度に関するよくある質問

Adobeデジタルエクスペリエンス認定制度プログラムについて詳しくは、以下のよくある質問（FAQ）を参照してください。

![バナー](/help/certifications/assets/home_banner_narrow.png)

>[!BEGINTABS]

>[!TAB  一般 ]

+++認定制度とは何ですか？また、メリットは何ですか？

個人のメリット：資格認定は、Adobe申請における個人のスキルとコンピテンシーを認定する資格です。 認定試験に合格した者は、Adobeの認定を受けた専門家またはマスターになります。

認定にはコミットメントが必要で、法的防御力があり、専門知識のAdobeによる検証が必要です。 [!DNL Credly] からバッジをダウンロードして、ソーシャルプロファイルで認定制度を機能させることもできます。

[!DNL Credly] でバッジを管理する方法について詳しくは、[ こちら ](https://experienceleague.adobe.com/docs/certification/certification/how-to-get-certified.html){target="_blank"} を参照してください。

+++

+++どの試験を受けるべきかどうすれば分かりますか。

資格認定は役割に基づいています。つまり、複数の試験が利用可能です。 資格認定ジャーニーページを確認して、受ける試験を選択します。 試験の選択方法：

* アプリケーション：例：[!DNL Adobe Analytics] または [!DNL Marketo Engage]
* 役割：例：開発者や業務担当者
* レベル：経験値 0～12 か月のプロフェッショナル、1～3 年のエキスパートなど

試験に複数のバージョンが存在する場合もあります。 最新バージョンを使用することをお勧めします。

* 試験が廃止される場合は、資格認定ジャーニーページにメモが表示されます。 これは、試験の新しいバージョンが利用可能であるため、この試験バージョンがすぐに削除されることを意味します。 最新バージョンを取得することをお勧めしますが、取得する準備が整ったら、廃止するバージョンを取得できます。 いずれにせよ、あなたの資格認定は 2 年間続きます。
* 現在の資格認定の有効期限が近づいている場合は、更新する必要があります。 更新オプションは、質問が少なく、無料です。 これらのオプションは、最新のアプリケーション機能に対応しています。 認定資格の有効期限が切れると、最新バージョンの認定試験を受ける必要があります。その場合、支払いが発生します。 [ 更新オプションの詳細情報 ](https://experienceleague.adobe.com/docs/certification/certification/renew.html){target="_blank"}。

+++

+++診察中に技術的な問題が発生した場合はどうすればよいですか？

[!DNL Examity] には、チャット、メール、電話によるサポートが用意されています。 Adobeサポートは、試験中に発生する技術的な問題の解決には役立ちません。

[!DNL Examity] サポート：`adobepartnersupport@examity.com`

最も迅速なヘルプリソースは、彼らのチャットを通じてです。 すべての受験者は、試験を受ける前に、コンピューターに適切なシステム要件があることを確認する必要があります。

詳しくは、試験と技術的な設定の管理に関する [!DNL Examity] のページを参照してください。

+++

+++認定制度に関する質問や問題に連絡できるのはだれですか？

試験登録、試験の支払い、テスト配信、テストセンター、試験に関するコメント、試験の質問、テスト経験、テスト宿泊施設、認証ポータルに関する質問については、`certif@adobe.com` にお問い合わせください。

+++

+++試験の予約をキャンセルしたり、スケジュールを変更したりできますか？

受験を予定されている 48 時間前までに開始した場合は、通常、キャンセルと再スケジュールが可能です。 候補者は、このアポイントメントを自分で変更する必要があります。

Adobeサポートまたは検収のどちらも、再スケジュールおよびキャンセルのリクエストを処理できません。 48 時間以内のキャンセルは、試験手数料に加えて再予定手数料の対象となります。

試験の 48 時間以上前であっても、試験のスケジュールを変更できない場合は、新しいバージョンの試験が存在する可能性があります。 アポイントメントが以前のバージョンの試験の場合、試験のアポイントメントのスケジュールを変更することはできません。 出来る限り、試験の予約は取っておくことをお勧めします。 できない場合は、試験をキャンセルし、新しい試験を受けるための新しいアポイントメントをスケジュールする必要があります。

+++

+++Adobeは特別な要件を持つ人材を受け入れることができますか？

Adobeは特定の特別な要件に対応できます。 試験を完了するために追加の時間が必要な場合や、他の宿泊施設を利用する場合は、リクエストに `certif@adobe.com` い合わせください。

**試験スケジュールの 2 週間前までにリクエストを送信する必要があります。**

+++

+++不合格になった場合、認定試験を再び受けることはできますか？

Adobe認定資格を取得または保持するには、再受験して最新の認定試験に合格する必要があります。 資格認定の有効期限が切れるまでに、潜在的なリテイクを考慮するために十分な時間を確保することをお勧めします。

1 回目でAdobe認定試験に落ちた場合は、少なくとも 24 時間待ってから再受験する必要があります。 受験に失敗した場合は、受験に再参加するまでに 14 日間の待機期間が必要です。 各回の試行には個別の試験料金が発生します。

+++

+++スコアレポートのコピーを取得するにはどうすればよいですか？

* 手順 1:[Adobe資格情報管理システム ](https://www.certmetrics.com/adobe/){target="_blank"}、CertMetrics にログインします。 ID またはメールアドレスとパスワードを使用してログインします。
* ステップ 2：履歴をクリックします。
* 手順 3：トランスクリプト履歴をスクロールし、表示したいレポートを選択します。 レポートはメールで送信することも、画面で表示することもできます。

+++

+++資格認定のステータスを追跡するにはどうすればよいですか？

[ 資格認定 ] タブの [Adobe資格情報管理システム ](https://www.certmetrics.com/adobe/){target="_blank"} に移動すると、資格認定の状態を管理できます。

+++

+++認定試験に合格すると、何が得られますか。

試験が完了すると、結果が直ちに画面に表示されます。 72 時間以内に、CertMetrics からスコアの確認メールが送信され、証明書がPDFされます。 CertMetrics アカウントから、資格認定のステータスを示すロゴにアクセスでき、Web サイト、電子メール、署名、名刺、ソーシャルプロファイルなどに使用できます。

+++

+++Adobe資格情報管理/Certmetrics アカウントにログインしたり、アカウントを作成したりするにはどうすればよいですか？

[ ログインするか、こちらからアカウントを作成 ](https://www.certmetrics.com/adobe/){target="_blank"} できます。 利用可能な試験、試験履歴、資格認定の有効期限、割引などの特典が表示されます。

+++

+++認定制度の期間はどのくらいですか？

Adobe認定を受けた方が新しいアプリケーションの機能に関する最新情報を入手できるようにする必要があるため、Adobe認定は **2 年間有効** です。

+++

+++英語以外の言語での試験はありますか？

日本語で試験を受けることもできます。 日本語で試験が可能な場合は、その試験の詳細に記載されます。 試験の詳細は、資格認定ジャーニーページの「準備」タブにあります。

+++

+++Adobe大会での資格認定は受けられますか？

認定制度は、Adobe主催のイベントで提供されます。

+++

+++認定制度は特定のバージョンのアプリケーション（Adobe Experience Manager 6.5 など）に関連付けられていますか？

直接ではありません。いいえ。 ただし、2 年ごとに更新が必要となるため、認定を受けることでAdobeアプリの最新版を確実に最新の状態に保つことができます。

+++

>[!TAB  オンデマンドテストによる更新 ]

+++認定更新とは？ 資格認定を更新するにはどうすればよいですか？

Adobe認定は 2 年後に失効します。 該当するAdobe認定試験の最新版を受けて合格することで、認定資格を更新できます。

次の 2 つのオプションがあります。

* 定期認定試験を受ける
* 更新試験を受けてください。より短い試験で、最新のソリューション機能に重点を置き、既に知っていることに焦点を当てます。

最初の認定資格を持っていない場合や、認定資格の有効期限が切れている場合は、更新試験を受けることはできません。

たまに役職試験を引退するんです。 この場合、認証は 2 年間有効になり、その後、認証の有効期限が切れます。 例えば、リード/開発業務認定を提供しなくなったので、交換試験を受けなくても認定が期限切れになります。

+++

+++有効期限が切れる前に認証を更新しない場合はどうなりますか？

資格認定は期限切れになり、[!DNL Credly] および任意のソーシャルプロファイルに期限切れと表示されます。 また、資格認定の有効期限が切れると、より短い更新試験を受けたり、継続的な学習によって更新したりできなくなります。

[!DNL Credly] でバッジを管理する方法について詳しくは、[ こちら ](https://experienceleague.adobe.com/docs/certification/certification/how-to-get-certified.html){target="_blank"} を参照してください。

+++

+++資格認定を更新するにはどの程度のコストがかかりますか？

無料更新が可能です。 これにより、コスト効率の高い方法で資格認定を維持し、ソリューションスキルを最新の状態に保つことができます。

資格認定の更新方法について詳しくは、[ こちら ](https://experienceleague.adobe.com/docs/certification/certification/renew.html){target="_blank"} を参照してください。

+++

+++資格認定の有効期限はどこで確認できますか？ 通知を受け取りましょうか。

[Adobe資格情報管理システム ](https://www.certmetrics.com/adobe/){target="_blank"} にサインインします。 証明書を取得した日時と、有効期限が切れる日時を確認できます。 また、プロファイルを確認し、連絡先、セカンダリメール、ジョブ情報が正しいことを確認する必要があります。

資格認定の有効期限が切れる 1 か月前にメール通知が届きます。 ただし、ジョブを変更してもメールを更新していない場合、通知が届かないことがあります。 CertMetrics アカウントにセカンダリメール（Gmail、Hotmail、Yahoo など）を追加して、会社を変更した場合でも認証の有効期限リマインダーメールを受信できるようにします。

+++

+++リニューアルにはどの試験を受ける必要がありますか。

取得が必要な試験を確認するには、Adobe IDを使用して [Adobe資格情報管理システムにログインし ](https://www.certmetrics.com/adobe/){target="_blank"} 「認定制度」をクリックして、認定名を選択します。 更新試験は **最新の状態を維持する要件** の下にリストされています。

+++

+++資格認定を更新すると、新しいデジタルバッジが届きますか？

いいえ。デジタルバッジは変わりません。

+++

+++資格認定を更新するために最初の資格認定で受けた試験と同じ試験を受けることはできますか？

いいえ、最初の資格認定に合格した後は、同じ試験を再受験して更新することはできません。

+++

>[!TAB  継続的な学習による更新 ]

+++継続的なラーニングを通じたAdobeの認定更新プログラムとは

Adobeでは、継続的な学習とエンゲージメントのアクティビティを提供しており、候補者がアクティブな状態を維持し、資格認定のステータスをさらに 2 年間延長できるようにします。 これは、一部のアプリケーションにのみ提供されます。

+++

+++資格認定は既に期限切れになっています。継続的な学習とエンゲージメントのアクティビティを更新のために続行できますか？

いいえ。この更新オプションは、Adobeでアクティブに認定されている Analytics、Experience Manager、Commerce、Target の候補者のみを対象としています。

+++

+++継続的な学習とエンゲージメントのアクティビティを更新できる認定制度はどれですか？

現在、このプログラムには、Analytics、Experience Manager、Commerceおよび Target が含まれています。 当社は、他の認定用に拡張する場合と同様に、該当するチャネルを通じてお知らせを共有します。

+++

+++認定更新プログラムには、どのような継続的な学習やエンゲージメントの活動が含まれていますか？

候補者は、レビューするために 5～7 のオンデマンドコースを提供されます。 認定制度を更新する基準は、コースまたはパス 2 の評価に基づいて 3 つの評価に合格し、署名済みのお客様参照フォームを送信することです。

+++

+++認定制度の更新に関する顧客参照フォームとは何ですか？

Adobeの継続的な学習およびエンゲージメントアクティビティの一環として、認定資格を更新するために候補者が送信できるフォーム。 このフォームには、お客様が署名し、候補者が過去 12 か月以内に、資格認定を更新する製品に関するプロジェクトに 100 時間以上を費やしたことを示す必要があります。 このオプションを資格認定の更新要件の一部に含めることにより、アドビでは、資格認定を引き続きアクティブな状態に保つためにスキルと知識を検証する方法として、継続的なエンゲージメントを認めたいと考えています。

+++

+++継続的な学習やエンゲージメントの活動を通じて資格認定の更新を実施する場合、料金はかかりますか？

いいえ。すべてのアクティビティは、Adobeの Credential Manager から無料でアクセスできます。

+++

+++評価に合格し、顧客紹介フォームも送信する必要がありますか？

いいえ。顧客参照フォームの送信はオプションであり、候補者が資格認定を更新するために完了できるアクティビティの 1 つです。 例えば、認証 [!DNL Adobe Analytics] 更新するには、特定のリストから少なくとも 3 つのアクティビティを完了する必要があります。 このリストには、6 つの評価を渡し、顧客参照フォームを送信することが含まれています。 候補者は、2 つの評価に合格し、顧客参照フォームを送信して、更新の最小 3 つの要件を満たすことができます。

+++

+++1 つのソリューションで 3 つの認定制度を保有しています。 各資格認定の要件を完了する必要がありますか？

いいえ、必要な操作は、ソリューションにリストされている要件だけです。 資格認定は、必要なアクティビティを完了し、結果をAdobeCredential Manager に送信した後に更新されます。 例えば、実務担当者、開発者およびアーキテクトとして [!DNL Adobe Analytics] で複数の認定資格を保持している場合、更新するすべての認定資格の更新要件を満たすために必要なアクティビティは、3 つだけです。

+++

+++コースを確認する必要がありますか。それとも、評価を受けるだけで開始できますか。

審査はExperience Leagueのコースに基づいており、審査を経て合格するための準備を整えます。 ただし、更新の重要な要件は、評価を合格することです。
